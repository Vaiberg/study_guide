# Шпаргалка по Python: Базовые Операции

## Базовый Уровень

### Математические операции

| Операция       | Символ   | Пример               | Описание                                    |
|----------------|----------|----------------------|---------------------------------------------|
| Сложение       | `+`      | `a + b`              | Сумма двух чисел                            |
| Вычитание      | `-`      | `a - b`              | Разность двух чисел                         |
| Умножение      | `*`      | `a * b`              | Произведение двух чисел                     |
| Деление        | `/`      | `a / b`              | Деление (результат всегда с плавающей запятой) |
| Целочисленное деление | `//` | `a // b`         | Деление с округлением вниз (целочисленное)  |
| Остаток от деления | `%`   | `a % b`              | Остаток от деления                          |
| Возведение в степень | `**` | `a ** b`            | Возведение числа `a` в степень `b`          |

### Логические операции

| Операция       | Символ   | Пример               | Описание                                    |
|----------------|----------|----------------------|---------------------------------------------|
| И              | `and`    | `a and b`            | Истинно, если оба выражения истинны         |
| ИЛИ            | `or`     | `a or b`             | Истинно, если хотя бы одно выражение истинно|
| НЕ             | `not`    | `not a`              | Истинно, если `a` ложно                    |

### Сравнительные операции

| Операция       | Символ   | Пример               | Описание                                    |
|----------------|----------|----------------------|---------------------------------------------|
| Равно          | `==`     | `a == b`             | Истинно, если `a` равно `b`                |
| Не равно       | `!=`     | `a != b`             | Истинно, если `a` не равно `b`             |
| Больше         | `>`      | `a > b`              | Истинно, если `a` больше `b`               |
| Меньше         | `<`      | `a < b`              | Истинно, если `a` меньше `b`               |
| Больше или равно | `>=`   | `a >= b`             | Истинно, если `a` больше или равно `b`     |
| Меньше или равно | `<=`   | `a <= b`             | Истинно, если `a` меньше или равно `b`     |

### Операции над строками (str)

| Операция                | Символ/Метод      | Пример                     | Описание                                              |
|-------------------------|-------------------|----------------------------|-------------------------------------------------------|
| Конкатенация            | `+`               | `"Hello" + "World"`         | Объединение строк                                      |
| Повторение              | `*`               | `"Hello" * 3`               | Повторение строки несколько раз                       |
| Длина строки            | `len()`           | `len("Hello")`              | Возвращает длину строки                               |
| Поиск подстроки         | `.find()`         | `"Hello".find("e")`         | Возвращает индекс первого вхождения подстроки         |
| Преобразование регистра | `.upper()/.lower()` | `"Hello".upper()`          | Преобразует строку в верхний/нижний регистр           |
| Разделение строки       | `.split()`        | `"Hello World".split()`     | Разбивает строку на список подстрок                   |

### Операции над списками (list)

| Операция                | Символ/Метод     | Пример                     | Описание                                              |
|-------------------------|------------------|----------------------------|-------------------------------------------------------|
| Конкатенация            | `+`              | `[1, 2] + [3, 4]`           | Объединение списков                                   |
| Повторение              | `*`              | `[1, 2] * 3`                | Повторение списка несколько раз                       |
| Длина списка            | `len()`          | `len([1, 2, 3])`            | Возвращает длину списка                               |
| Добавление элемента     | `.append()`      | `lst.append(4)`             | Добавляет элемент в конец списка                      |
| Вставка элемента        | `.insert()`      | `lst.insert(1, "Hello")`    | Вставляет элемент на указанную позицию                |
| Удаление элемента       | `.remove()`      | `lst.remove(2)`             | Удаляет первое вхождение элемента                     |
| Удаление по индексу    | `.pop()`         | `lst.pop()`                 | Удаляет и возвращает последний элемент                |
| Удаление по индексу     | `.pop(index)`    | `lst.pop(0)`                | Удаляет и возвращает элемент по индексу               |
| Сортировка списка       | `.sort()`        | `lst.sort()`                | Сортирует список по возрастанию                       |
| Обратная сортировка     | `.reverse()`     | `lst.reverse()`             | Переворачивает список                                 |

### Операции над кортежами (tuple)

| Операция                | Символ/Метод     | Пример                     | Описание                                              |
|-------------------------|------------------|----------------------------|-------------------------------------------------------|
| Конкатенация            | `+`              | `(1, 2) + (3, 4)`           | Объединение двух кортежей                             |
| Повторение              | `*`              | `(1, 2) * 3`                | Повторение кортежа несколько раз                      |
| Длина кортежа           | `len()`          | `len((1, 2, 3))`            | Возвращает количество элементов в кортеже             |
| Доступ по индексу       | `[]`             | `tup[1]`                    | Доступ к элементу по индексу                          |
| Проверка элемента       | `in`             | `1 in (1, 2, 3)`            | Проверяет наличие элемента в кортеже                  |

### Операции над множествами (set)

| Операция                | Символ/Метод      | Пример                     | Описание                                              |
|-------------------------|-------------------|----------------------------|-------------------------------------------------------|
| Объединение множеств     | `|`               | `{1, 2} | {3, 4}`            | Объединение двух множеств                             |
| Пересечение множеств     | `&`               | `{1, 2} & {2, 3}`           | Пересечение двух множеств                             |
| Разность множеств        | `-`               | `{1, 2} - {2, 3}`           | Разность двух множеств                                |
| Симметрическая разность  | `^`               | `{1, 2} ^ {2, 3}`           | Элементы, которые есть только в одном из множеств     |
| Добавление элемента      | `.add()`          | `set.add(5)`                | Добавление элемента во множество                      |
| Удаление элемента        | `.remove()`       | `set.remove(2)`             | Удаление элемента из множества                        |

### Операции над словарями (dict)

| Операция                | Символ/Метод      | Пример                     | Описание                                              |
|-------------------------|-------------------|----------------------------|-------------------------------------------------------|
| Получение значения по ключу | `[]`         | `dct["key"]`                | Получение значения по ключу                           |
| Добавление/Изменение пары | `[]=`         | `dct["new_key"] = value`     | Добавление новой или изменение существующей пары      |
| Удаление элемента       | `.pop()`          | `dct.pop("key")`            | Удаляет и возвращает значение по ключу                |
| Ключи словаря           | `.keys()`         | `dct.keys()`                | Возвращает все ключи словаря                          |
| Значения словаря        | `.values()`       | `dct.values()`              | Возвращает все значения словаря                       |
| Пары ключ-значение      | `.items()`        | `dct.items()`               | Возвращает список пар ключ-значение                   |

### Специальные символы

| Символ       | Описание                                       |
|--------------|------------------------------------------------|
| `#`          | Комментарий, текст после него игнорируется      |
| `"` или `'`  | Описание строки (одинарные и двойные кавычки)   |
| `\\`         | Экранирование символа (например, `\"` для кавычек) |
| `\n`         | Перевод строки                                 |
| `\t`         | Табуляция                                      |

## Продвинутый Уровень

### Генераторы и `yield`

| Операция                | Пример                     | Описание                                              |
|-------------------------|----------------------------|-------------------------------------------------------|
| Создание генератора      | `def gen(): yield x`       | Определяет генераторную функцию                       |
| Получение следующего значения | `next()`             | Возвращает следующее значение из генератора            |

### Операции с `zip()`

| Операция                | Пример                     | Описание                                              |
|-------------------------|----------------------------|-------------------------------------------------------|
| Объединение списков      | `zip([1, 2], [3, 4])`      | Возвращает итератор пар: `[(1, 3), (2, 4)]`           |

### Операции с `enumerate()`

| Операция                | Пример                     | Описание                                              |
|-------------------------|----------------------------|-------------------------------------------------------|
| Перебор списка с индексами | `enumerate(lst)`         | Возвращает итератор пар `(индекс, значение)`           |

### Списковые включения (List comprehension)

| Операция                | Пример                     | Описание                                              |
|-------------------------|----------------------------|-------------------------------------------------------|
| Создание списка          | `[x**2 for x in range(5)]` | Создаёт список квадратов чисел `[0, 1, 4, 9, 16]`     |
| С фильтрацией элементов  | `[x for x in range(10) if x % 2 == 0]` | Только чётные числа `[0, 2, 4, 6, 8]` |
